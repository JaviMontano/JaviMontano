### How It Works

Every framework follows the same core principles:

**Zero-Hallucination Protocol** вҖ” Every claim tagged with evidence: `[CODE]` `[CONFIG]` `[DOC]` `[INFERENCE]` `[ASSUMPTION]`. If >30% assumptions, mandatory warning.

**Quality Gates** вҖ” No phase advances without validation. Hard stops at G0 (security), G1, G1.5, G2, G3.

**MOAT Pattern** вҖ” Each skill ships with Methods, Ontology, Assets, and Templates. Progressive loading (L1/L2/L3) for context efficiency.

**Fundamentar antes de acelerar.** Ground before you accelerate.
